# YOLOv9-Traffic-Signs-Detection
Trained a YOLOv9-E model to classify 15 types of traffic signs, achieved high accuracy for real-time traffic sign recognition.

Link for the Kaggle notebook: https://www.kaggle.com/code/majdmrawed/yolov9-traffic-signs-detection

The Dataset used contains 3530 images for training, 801 for validation, and 638 for testing. There are 15 classes for classification.: https://www.kaggle.com/datasets/pkdarabi/cardetection
